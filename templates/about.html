Arrests Demographics in NYC

{% include 'navbar.html' %}
Every year, thousands of people are arrested in NYC alone. This project seeks to map the arrests that occurred by NYPD in 2023, providing details about the perpetrator and the crime. Using this data, and with the help of infographics, an accurate representation of crime rates in each borrow can be conveyed and compared.

Data cleansing

  1. A data set was downloaded from NYC OpenData.
  2. Google sheets was used to remove columns of unneccessary data that the general public would not understand, like law codes.
  3. A useful dicionary-based data structure was designed in order to make all necessary data organized and accessible.
  4. A python program (format_data.py) was created to read the .csv line-by-line to create and save the dictionary as a .json file.
